# ServerFileManager

A straightforward and automated file sorting utility. It scans your chaotic directories and automatically moves files into categorized folders, so you can easily drop the clean structure directly onto your server. No bullshit, just sorted files.

## Supported Categories & File Types

## Why?
Built to automate the annoying process of manually sorting new files. Run the program, let it sort everything into the predefined folders, and copy the organized data directly to your server. gg.
